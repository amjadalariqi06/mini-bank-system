## 🇬🇧 English Version

### 📝 Project Description
This project is a console-based **Mini-Banking System Simulation**. The primary focus was optimizing the core business logic, debugging complex data flows, and implementing strict security boundaries for financial transactions. It provides a robust, crash-free simulation of everyday banking operations.

### 🚀 Key Features
* 👤 **Account Management:** User authentication and creation of secure, unique account profiles.
* 💸 **Core Transactions:** Full support for seamless Deposits, Withdrawals, and Fund Transfers.
* 📄 **Transaction History:** Real-time logging of all financial activities for accurate account statements.

### 🛠️ My Key Contributions & Fixes
* **Robust Exception Handling:** Implemented comprehensive validation checks to handle invalid or malicious inputs gracefully, preventing system crashes.
* **Logic Correction:** Resolved logical bugs related to over-drafting (preventing withdrawal beyond the available balance) and restricted negative financial inputs.
* **Data Integrity:** Ensured safe operations during fund transfers, meaning both the sender’s and receiver’s balances update concurrently without data mismatch or loss.

### 💻 Technologies Used
* **Language:** C++ / Object-Oriented Programming (OOP)
* **Data Management:** File Handling (TXT / Binary files) for permanent data storage.

## 🇸🇦 النسخة العربية

### 📝 وصف المشروع
هذا المشروع عبارة عن **محاكاة لنظام بنكي مصغر**. كان التركيز الأساسي فيه منصباً على تحسين وتطوير منطق العمليات (Business Logic)، وإصلاح الثغرات البرمجية في تدفق البيانات، وتطبيق قيود صارمة لضمان أمان التعاملات المالية. يوفر النظام تجربة مستقرة ومحمية ومحاكية للأنظمة البنكية الحقيقية.

### 🚀 الميزات الأساسية
* 👤 **إدارة الحسابات:** إنشاء حسابات جديدة برقم فريد والتحقق الآمن من هوية المستخدمين.
* 💸 **العمليات المالية:** دعم كامل وعملي لعمليات الإيداع، السحب، والتحويل المالي بين الحسابات المختلفة.
* 📄 **كشف الحساب:** توثيق وتسجيل لحظي لجميع العمليات لتوليد كشوفات حساب دقيقة وموثوقة.

### 🛠️ أبرز الإصلاحات والتحسينات التي قمت بها
* **معالجة الاستثناءات (Exception Handling):** إضافة شروط وقائية صارمة للتحقق من المدخلات ومنع النظام من الانهيار (Crash) عند إدخال بيانات خاطئة.
* **إصلاح منطق العمليات:** معالجة المشاكل البرمجية المتعلقة بالسحب بالسالب (منع سحب مبالغ أكبر من الرصيد المتاح) ومنع إدخال أي قيم سالبة أثناء الإيداع.
* **سلامة وثبات البيانات (Data Integrity):** ضمان تحديث أرصدة الحسابات بدقة متناهية بالتزامن أثناء عمليات التحويل لضمان عدم ضياع أو تضارب البيانات المالية.

### 💻 التقنيات المستخدمة
* **اللغة البرمجية:** C++ / البرمجة كائنية التوجه (OOP).
* **إدارة البيانات:** التعامل مع الملفات (File Handling - TXT/Binary) لتخزين وحفظ بيانات الحسابات بشكل دائم.
